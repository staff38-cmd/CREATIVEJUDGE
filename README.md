# CREATIVEJUDGE

薬機法・景品表示法などの広告規制に対し、クリエイティブ素材の適合性を Claude AI が自動チェックするプラットフォームです。

## 主な機能

- **マルチコンテンツ対応** — 画像・動画・LP・テキスト原稿・PDFに対応
- **AI規制チェック** — 薬機法・景品表示法・健康増進法・広告ガイドラインをClaude AI（claude-opus-4-6）が自動チェック
- **画像ビジョン解析** — 画像広告はビジョンAIで文字・ビジュアルを直接解析
- **詳細レポート** — 違反・警告・注意の3レベルで問題箇所を引用して指摘、改善案も提示
- **カスタムレギュレーション** — クライアント独自の禁止事項・プラットフォームポリシーを追加指定可能

## チェック対象規制

| 規制 | 主なチェック内容 |
|------|---------------|
| 薬機法（68条等） | 未承認効能効果の標榜、化粧品の医薬品的表現 |
| 景品表示法 | No.1・最高表示、優良誤認・有利誤認表示 |
| 健康増進法 | 誇大広告、消費者を誤認させる健康効果の表現 |
| 医師法・医療法 | 診断・治療・処方を示唆する表現 |
| 広告ガイドライン | ビフォーアフター誇張、根拠なき体験談 |

## セットアップ

```bash
# 依存関係をインストール
npm install

# 環境変数を設定
cp .env.example .env.local
# .env.local に ANTHROPIC_API_KEY を設定してください

# 開発サーバーを起動
npm run dev
```

[http://localhost:3000](http://localhost:3000) を開いてください。

## 技術スタック

- **フレームワーク**: Next.js 15 (App Router)
- **言語**: TypeScript
- **スタイリング**: Tailwind CSS v4
- **AI**: Anthropic Claude API (claude-opus-4-6)
- **ストレージ**: JSONファイル（`data/works.json`）
- **ファイル**: ローカル保存（`public/uploads/`）

## ページ構成

| パス | 説明 |
|------|------|
| `/` | トップページ（対応規制一覧・使い方） |
| `/submit` | コンテンツ登録（ファイル or テキスト貼り付け） |
| `/works` | チェック履歴一覧（種別・状態フィルター） |
| `/works/[id]` | コンプライアンスレポート詳細 + 再チェック |

## API

| メソッド | パス | 説明 |
|----------|------|------|
| GET | `/api/works` | チェック履歴一覧取得 |
| POST | `/api/works` | コンテンツ登録（FormData or JSON） |
| GET | `/api/works/[id]` | 詳細取得 |
| POST | `/api/analyze` | AIチェック実行 |
